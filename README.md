# MRKER-BOT

🤖 A powerful WhatsApp Multi-Device bot built for speed, reliability, and advanced automation.

## Features

- AI Auto Reply
- Auto Status View
- Auto Status React
- YouTube Downloads
- TikTok Downloads
- Sticker Creation
- Group Management
- Admin Commands
- Fast and Reliable Performance

## Installation

```bash
git clone <repository-url>
cd mrker-bot
npm install
```

## Start the Bot

```bash
npm start
```

## Requirements

- Node.js 18+
- WhatsApp Multi-Device
